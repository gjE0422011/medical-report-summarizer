# 🏥 Medical Report Summarizer

AI-powered app that reads medical PDF reports and explains 
them in plain English using Mistral 7B via Hugging Face.

**Live App:** [your-link-here]

## Features
- Upload any medical PDF
- Plain-English summary with key findings
- Flags abnormal values
- Suggests next steps
- Download summary as text file

## Stack
- Python · Mistral 7B · Hugging Face Inference API · PyMuPDF · Streamlit

## Run Locally
pip install -r requirements.txt
streamlit run app.py
```

## Step 5 — Deploy on Streamlit Cloud

1. Push all 3 files to GitHub
2. Go to **share.streamlit.io** → **New app**
3. Select your repo → main file: `app.py`
4. Click **Deploy**
```
